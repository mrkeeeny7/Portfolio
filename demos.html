OpenGL ES Demos

Shader Views

Longstanding and popular ShaderViews demo - one of the first demos for mobile GPU showcasing programmable shaders and render-to-texture (in this case displaying a variety of possible post-processing effects). Uses OpenGL ES 2.0

See also this link for the demo usage in context, running on embedded hardware.

Leaves

One of the first demos to use OpenGL ES 3.0; newly available instancing and transform feedback enabled much of the work animating and displaying the individual leaves to be loaded onto the GPU.

Being demoed at GDC 2013: See https://www.youtube.com/watch?v=1T7z2_OcFDs

Shader Show

The first demo showcasing multiple programmable shaders on newly OpenGL ES 2.0-capable hardware.
